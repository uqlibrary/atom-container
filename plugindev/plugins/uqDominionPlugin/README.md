# arFryerPlugin
This plugin is based on the uqDominionPlugin, which is the default official theme for atom.
It is available [here](https://github.com/artefactual/atom/tree/stable/2.4.x/plugins/uqDominionPlugin).

Changes pushed to master in this repository will be synced to atom, built and enabled.

# Manual Changes
In order to finish the desired look we're going for with this theme, go ahead and visit the following settings pages within Atom.

## Default Page Elements
* Logo - Tick
* Title - Tick
* Description - Untick
* Language menu - Untick
* Digital object carousel - Tick
* Digital object map - Untick
* Copyright status filter - Tick
* General material designation filter - Tick

## User interface labels
* globalSearch = Search manuscripts
